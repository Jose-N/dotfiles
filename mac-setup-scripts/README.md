# MAC Setup Instructions

Order of scripts to run when starting a fresh install of a mac, maybe automate this at some point.

* Run ./install-homebrew.sh
* Restart terminal
* Run ./install-apps.sh
* Run ./install-configs.sh
* Run `mise i` from `$HOME`
* Restart terminal
* Run ./setup-langs.sh
* Run `doom sync`
* Run `doom doctor`
